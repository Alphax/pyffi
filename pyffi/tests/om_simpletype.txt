Regression tests for L{PyFFI.ObjectModels.SimpleType}
=====================================================

Constructor value argument
--------------------------

>>> from PyFFI.ObjectModels.SimpleType import SimpleType
>>> test = SimpleType()
>>> print(test)
None
>>> test = SimpleType(value='eek!')
>>> test.value
'eek!'

Interchangeability
------------------

>>> test1 = SimpleType(2)
>>> test2 = SimpleType(2)
>>> test1 is test2
False
>>> test1.isInterchangeable(test2)
True
>>> test2.value = 'hello'
>>> test1.isInterchangeable(test2)
False
