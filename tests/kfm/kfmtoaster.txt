Doctests for the kfmtoaster script
==================================

The --spells switch
-------------------

>>> import sys
>>> sys.path.append("scripts/KFM")
>>> import kfmtoaster
>>> sys.argv = ["kfmtoaster.py", "--spells"]
>>> kfmtoaster.KfmToaster().cli()
check_read
check_readwrite
dump
