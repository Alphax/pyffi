Doctests for the kfmtoaster script
==================================

The --spells switch
-------------------

>>> import sys
>>> sys.argv = ["kfmtoaster.py", "--spells"]
>>> import PyFFI.Spells.KFM
>>> PyFFI.Spells.KFM.KfmToaster().cli()
check_read
check_readwrite
dump
